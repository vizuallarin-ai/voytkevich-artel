# Этап 7 — Доработка планировщика дома

Сайт: [voytkevich-artel.vercel.app](https://voytkevich-artel.vercel.app/)  
Дата: июнь 2026

---

## 1. Что изменено

Страница `/planirovka` превращена в пошаговый планировщик-лидогенератор: 8 сценариев жизни, редактор комнат, сводка площади, рекомендации, похожие проекты, связь с калькулятором и расширенная форма заявки.

---

## 2. Структура страницы

1. Breadcrumbs  
2. Hero (`PlannerHero`)  
3. Funnel hint  
4. Пошаговый wizard (5 шагов)  
5. FAQ  
6. SEO-текст  
7. Sticky summary (mobile, шаг 4+)

**Шаги wizard:** Сценарий → Дом и участок → Комнаты → Сводка → Заявка

---

## 3. Сценарии жизни (8)

Семья с детьми, переезд из квартиры, дом для пары, для родителей, дачный, постоянное проживание, с кабинетом, с террасой.

Каждый сценарий задаёт: площадь, этажность, спальни/санузлы, список помещений с площадями по умолчанию.

---

## 4. Логика площади

`src/lib/planner-area.ts`:

- Жилая, техническая, санузлы, проходные, дополнительная  
- Общая сумма vs целевой ориентир  
- Статусы: в пределах / запас / превышена / компактная  

Исправлено: сумма не показывает 0 при наличии комнат (safeArea, фильтрация NaN).

---

## 5. Рекомендации

Правила на основе сценария, жильцов, состава комнат, этажности, уклона участка, расхождения площади.

---

## 6. Связь с каталогом

`findPlannerProjects()` — до 6 проектов по площади ±30 м², этажности, террасе, гаражу, кабинету.

---

## 7. Связь с калькулятором

`buildPlannerCalculatorUrl()` → `/calculator?source=planner&area=...&purpose=...`

Контекст комнат сохраняется в `sessionStorage` (`planner-calc-context`).

---

## 8. Форма заявки

`PlannerLeadSection` → `LeadForm` с `source=planner`.

Payload в comment через `plannerSummaryForLead()`: сценарий, площади, комнаты, рекомендации, похожие проекты.

TODO: структурированные поля в API `/api/leads`.

---

## 9. Сохранение черновика

`localStorage` ключ `planner-draft-v1`, версия схемы `1`.

Кнопка «Сбросить планировку» очищает черновик.

---

## 10. Аналитика

`trackPlannerEvent()` — started, scenario_selected, step_completed, room_added/removed, area_changed, recommendations_viewed, calculator_clicked, project_clicked, reset.
